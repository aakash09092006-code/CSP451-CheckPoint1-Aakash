# CSP451 CheckPoint 1

This project was created as part of the CSP451 course to learn the basics of Git and GitHub. 
It demonstrates how version control is used to track changes in files and manage a project repository.

## Project Description

The repository contains a simple web project with HTML, CSS, and JavaScript files. 
The purpose of this project is to practice creating a Git repository, making commits, 
using branches, and working with pull requests on GitHub.

## Installation

1. Clone the repository from GitHub.

```bash
git clone https://github.com/aakash09092006-code/CSP451-CheckPoint1-Aakash.git
```

2. Open the project folder on your computer.

```bash
cd CSP451-CheckPoint1-Aakash
```

3. Open the `index.html` file in a web browser.

## Usage

This project is used for educational purposes only. 
It helps students understand Git commands, commit history, and GitHub workflows such as branching and pull requests.

Useful Git commands:

```bash
git log --oneline --graph --all
```

```bash
git diff
```

```bash
git status
```
